Request data deletion

App: Pass Gen
Android package: com.luckytools.pass_gen
Developer contact: autoaccentsni@gmail.com
Last updated: 17 July 2026

Use this page for Google Play’s data deletion request URL. Pass Gen has no accounts, no sign-in, and no in-app forms that send data to us. Password generation stays on your device. The only personal data the developer may hold is if you choose to email support.

What data exists where

  • On your phone (app storage) — generated passwords/history, settings, and a local Premium flag. We cannot access this remotely.
  • With the developer — only support emails you send us (and our replies). There is no Pass Gen cloud account or password database.
  • With Google / ad partners — Play Billing and AdMob process data under their own policies. We cannot delete those records for you; use Google’s tools (below).

Steps to delete your data

  1. Delete local app data (immediate — no email needed)
    On Android:
    • Open Settings → Apps → Pass Gen → Storage
    • Tap Clear storage / Clear data
    Or uninstall Pass Gen. This removes local history, settings, and the cached Premium flag for that install. No confirmation from us is required because we never held that data on a server.
  2. Clear history inside the app (optional)
    Open Pass Gen → History → clear, if you only want to wipe stored generations without uninstalling.
  3. Support emails only (the part we can delete for you)
    If you have emailed us and want that correspondence removed:
    • Email autoaccentsni@gmail.com with subject Pass Gen data deletion request.
    • Include the email address you used to contact us (and approximate date if you know it) so we can find the thread.
    What we will do: delete support messages we hold that we can identify as yours, within 90 days of a clear request (usually much sooner).
    Confirmation email: yes — we will send one short reply confirming that we have completed (or already had nothing to delete). That is the only automated-style notice; there is no separate account dashboard.
    If you never emailed us, there is nothing for us to delete on our side after you clear the app.
  4. Premium subscription (Google Play, not us)
    Premium is a Google Play subscription. Uninstall does not cancel billing — cancel under Google Play → Payments & subscriptions. Purchase records stay with Google. After reinstall, use Restore purchases in the app with the same Google account if you still want Premium.
  5. Advertising partners
    Free-tier ads use Google AdMob (and mediation partners where configured). Reset or limit your advertising ID in Android settings. See Google’s ad settings for further controls.

Google Play Console (for the developer)

Because Pass Gen has no user accounts, account-deletion in-app paths do not apply. For Data safety / data deletion answers:

  • App does not allow users to create an account
  • Users can request deletion — yes (this URL + clear storage / uninstall)
  • Developer-held data (support email only): process deletion requests within 90 days — you may select that option in Play Console

What we do not store

We do not run a Pass Gen cloud vault. Generated passwords are not uploaded by the app. We cannot delete a password from “our servers” because it was never sent to us.
